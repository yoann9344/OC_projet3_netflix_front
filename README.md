# OC_projet3_netflix_front
Netflix frontend without any framework (must use only HTML, CSS and JS)


## Usage
```bash
# create and go to your venv
cd api/
python manage.py runserver&
cd ../front/
npx rollup -c -w&
cd ../dist
python -m http.server 8080&
```
